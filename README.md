# 小程序滑动
小程序毕竟不是基于普通浏览器，普通的滑动组件无法在小程序中使用。<br/><br/>
官方的swiper功能有限，可改动局限太大（几乎改不动）。<br/><br/>
这里使用了[we-swiper](https://github.com/we-plugin/we-swiper)，改造成了这样一个首页，后面有时间想做个这样单一功能的极简组件，好像还是有需求的。<br/><br/>
